
Proyecto: Gestor de usuarios — listo para ejecutar (no requiere cambios)

Contenido:
- src/gestor/*.java
- run.sh (Linux/macOS) -> compila y ejecuta
- run.bat (Windows) -> compila y ejecuta
- docs/planificacion.pdf (breve planificación)

Instrucciones rápidas:
Linux/macOS:
  ./run.sh

Windows:
  run.bat

También puedes abrir directamente la carpeta en IntelliJ (File -> Open) y ejecutar la clase gestor.MainLauncher.
